1. Introduction to Conditionals in Arduino

What are Conditionals?
In programming, conditionals let your program make decisions. This means it can do different things based on different inputs. Think of it like: ‚ÄúIf it‚Äôs raining, use an umbrella.‚Äù

In Arduino, conditionals help you respond to real-world inputs.

Let's say you want to turn on an LED if it's dark: 


        // Read the light sensor value
        int lightValue = analogRead(A0);
        
        if (lightValue < 300) {
          digitalWrite(LED_BUILTIN, HIGH); // turn on LED
        } else {
          digitalWrite(LED_BUILTIN, LOW);  // turn off LED
        }

In the code above:

  • analogRead(A0) reads the value from the light sensor
  • If the light is below 300 (dark), the LED is turned on
  • Otherwise, it's turned off
