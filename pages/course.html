1. Introduction to conditionals

CONDITIONALS (OR SELECTION):
Selection statements (sometimes called conditional statements) are code that is executed only when a certain condition is satisfied. 

if (someCondition) {
  // do stuff if the condition is true
}

if (someCondition) {
  // do stuff if the condition is true
} else {
  // do other stuff
}
